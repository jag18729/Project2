# Project 2 - Chrome DevTools Examples

## Files

**Demo Pages:**
- `devtools-javascript.html` - JavaScript debugging examples
- `devtools-dom.html` - DOM manipulation examples

**Original Project:**
- `index.html` - Giga Pet (original Project 2)

## What to Do

1. **Open the demos:**
   - Open `devtools-javascript.html` in Chrome
   - Open `devtools-dom.html` in Chrome

2. **Take screenshots:**
   - Open each page and press F12
   - Take 10 screenshots (one for each example)
   - Save to `assets/` folder

3. **Create PDF:**
   - Open `assets/DevTools-Examples-Documentation.md`
   - Copy to Word/Google Docs
   - Add your 10 screenshots
   - Save as `DevTools-Examples-Documentation.pdf` in `assets/` folder

4. **Push to GitHub:**
   ```bash
   git add .
   git commit -m "Add DevTools examples"
   git push
   ```

Done!
